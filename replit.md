# Система гарантийных талонов SONMAR

## Обзор проекта
Современная веб-платформа для управления гарантийными талонами насосов SONMAR. Приложение построено на Next.js с использованием Tailwind CSS 4.0 для стилизации.

## Последние изменения (31 октября 2025)
- ✅ Создана красивая современная главная страница в стиле бренда SONMAR
- ✅ Настроен Tailwind CSS 4.0 с @tailwindcss/postcss
- ✅ Добавлен логотип компании в header и footer
- ✅ Реализован адаптивный дизайн с градиентами и анимациями

## Структура проекта
```
/pages
  - index.tsx - Главная страница с брендингом SONMAR
  - buyer.tsx - Страница покупателя
  - seller.tsx - Страница продавца
  - installer.tsx - Страница монтажника
  - admin.tsx - Панель администратора
  - login.tsx - Страница входа
  /api
    - login.ts - API для аутентификации

/styles
  - globals.css - Глобальные стили с Tailwind CSS

/public
  - sonmar-logo.webp - Логотип компании SONMAR
```

## Технологии
- **Next.js 15.2.3** - React-фреймворк для server-side rendering
- **React 19.0.0** - Библиотека для построения UI
- **Tailwind CSS 4.0.15** - Утилитарный CSS-фреймворк
- **TypeScript** - Типизация кода
- **PostCSS** - Обработка CSS

## Дизайн и брендинг
- **Цветовая палитра**: Оттенки синего и циана (в соответствии с брендом SONMAR)
- **Шрифты**: System fonts (San Francisco, Segoe UI, Roboto)
- **Компоненты**: Карточки с hover-эффектами, градиентные фоны, адаптивная навигация

## Запуск приложения
Приложение настроено на автоматический запуск через workflow "dev":
```bash
npm run dev -- --port 5000 --hostname 0.0.0.0
```

Сервер доступен на порту 5000 для просмотра в веб-интерфейсе Replit.
